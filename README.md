# KisanPushti AI Crop Doctor - Mobile App (React Native & Expo)

## How to Demo

1. Clone the repo:  
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
2. Install dependencies:  
npm install
3. Start Expo:
npm expo start
4. **Scan the QR code below** with the official **Expo Go app** (Android/iOS).  
- ![Expo QR Code](./path-to-your-qr-code.png)

5. App will launch instantly on your mobile!

## Alternate: APK Build (optional)
- Download APK from [https://drive.google.com/file/d/1yIko04OAYBdQLeM1KVpNyzlnucVaAWbr/view?usp=drivesdk] if available.
- Install directly on any Android device.

## Troubleshooting
- If QR fails, judges can run `npx expo start` and re-scan their own local QR.

## Features
- Multilingual crop disease reporting
- Voice assistant
- Scheme and price integration
